# Editorial Portfolio

Static portfolio site with an editorial, earthy visual direction and scroll-linked transitions.

## Preview locally

```bash
npm start
```

Open `http://localhost:4173`.

## Edit quickly

- Update copy in `index.html`
- Adjust palette and spacing in `styles.css`
- Fine-tune motion in `script.js`

## Deploy instantly

This site is plain HTML, CSS, and JavaScript, so you can drag the folder into Netlify, import it into Vercel, or publish it with GitHub Pages without a build step.
